# Benchmark CSV Results

This folder is where the computer saves the raw numbers from tests. 

* **benchmark_results.csv**: This has the speed and memory numbers for both the CPU and the different GPU versions (Naive, Tiled, and Fused).
* **cpu_benchmark.csv**: This has the detailed numbers just for the CPU baseline test.

I used these CSV files to draw the graphs that show which version of the attention code is the fastest.
